# 🚀 GrowX SMM Panel

GrowX is a lightweight and fast Social Media Marketing Panel UI built using pure HTML, CSS, and JavaScript.

This project is designed for fast deployment on GitHub Pages and works with Google Sheets + Apps Script API as backend.

---

✨ Features

- 🔐 Secure Login System
- 👤 User Registration System
- 💰 Add Fund Page
- 📦 Order Page
- 📊 Order History Page
- ⚙️ Settings Page
- 👑 Admin Panel

---

⚡ Tech Stack

- HTML5
- CSS3
- JavaScript
- Google Sheets Database
- Google Apps Script API

---

📂 Project Structure

index.html → Login / Signup
home.html → Dashboard
order.html → Place Order
addfund.html → Add Balance
history.html → Order History
setting.html → Account Settings
customer.html → Customer Support
gxadmin.html → Admin Panel
